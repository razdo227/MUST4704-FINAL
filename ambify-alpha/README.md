# ambify-alpha

Low-fidelity React/Next.js prototype for the ambify granular pad UI. This is layout-only (no DSP/audio yet) with sliders, toggles, and preset/export stubs.

## Run it

```bash
cd ambify-alpha
npm install
npm run dev
```

Open http://localhost:3000 to view the prototype.

## Lint (optional)

```bash
npm run lint
```

## Edit points

- UI and interactions: `src/app/page.tsx`
- Global styling: `src/app/globals.css`
